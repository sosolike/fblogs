1：查看文件所有内容:(cat)
    cat [-nbs] 文件路径
    功能：打印文件所有内容
    参数：
        -n：从1开始对每一行编号
        -b：除过空白行以外，从1开始对每一行编号
        -s：打印文件内容时，需要两个及以上的空白行换成一行空白行
    常见使用方式：
        1：查看文件全部内容：
            查看单个文件：cat 文件路径
            查看多个文件：cat 文件路径1 文件路径2
        2：查看文件内容，带上行号：
            cat -n 文件路径
        3：查看文件内容，除过空白行以外均带上行号：
            cat -b 文件路径
        4：将文件内容输出至另一个文件：
            cat 文件路径 > 目标输出文件路径

2：反向查看文件所有内容:(tac)
    tac 文件路径
    功能：反向打印文件所有内容
    参数：无
    常见使用方式：
        1：反向查看文件内容：
            tac 文件路径

3：查看文件内容并携带行号:(nl)
    nl [-bnw] 文件路径
    功能：查看文件内容，并带上行号
    参数：-b a：无论是否空行，都将列出行号(默认状态)
         -b t：除过空行以外，其余行都列出行号
         -n ln：指定行号位置在最左侧显示，并且不以0为前缀，空行无行号；
         -n rn：行号在屏幕左侧偏内容处显示，不以0为前缀，空行无行号；
         -n rz：行号在自己栏位的左侧偏内容处显示，并且以0为前缀，空行无行号；
    常见使用方式：   
        1：查看文件内容，并为所有行(包括空行)带上行号:
            nl 文件路径
            nl -b a 文件路径
        2：查看文件内容，空行不带行号:
            nl -b t 文件路径
        3：查看文件内容，并设置行号位置以及行号的格式：
            行号在内容开头左侧，无0为前缀：nl -n ln 文件路径
            行号在内容开头，无0为前缀：nl -n rn 文件路径
            行号在内容开头，并以0为前缀：nl- n rz 文件路径

4：查看文件内容前几行:(head)
    head [-ncqv] 文件路径
    功能：查看文件内容前几行，默认是前10行，该命令也可紧跟管道命令 (|)之后，用来查看其他命令的标准输出内容；
    参数：-n m：查看前m行
         -c m：查看前m个字节
         -q：隐藏文件名
         -v：显示文件名
    常见使用方式：
        1：查看前5行：
            head -n 5 文件路径
        2：查看前5个字节：
            head -c 5 文件路径

5：查看文件内容末尾几行:(tail)
    tail [-n num] 文件路径
    功能：查看文件末尾几行内容，并支持阻塞式查看，该命令也可紧跟管道命令(|)之后，用来查看其他命令的标准输出内容；
    参数：-n 行数
    常见使用方式：
        1：非租塞式查看文件末尾几行：
            tail -n 行数 文件路径
            tail -n行数 文件路径
        
        2：阻塞式查看文件末尾几行：
            tail -行数f 文件路径      (注意：阻塞式查看时，不能带-n参数，并且行数后要跟f)

6：翻页查看文件内容:(less)
    less 文件路径
    功能：翻页查看文件内容，该命令也可紧跟管道命令(|)之后，用来查看其他命令的标准输出内容；
    使用方式：
        [page up]：向上翻页
        [page down]：向下翻页
        空白键：向下翻页
        上箭头：向上翻一行
        下箭头：向下翻一行
        /子串：向下搜索子串内容
        ?子串：向上搜索子串内容
        n：重复前一个搜索
        N：反向重复前一个搜索
        q：退出less程序

7：翻页查看文件内容:(more)
    more 文件路径
    功能：翻页查看文件内容，该命令也可紧跟管道命令(|)之后，用来查看其他命令的标准输出内容；
    使用方式：
        空白键：向下翻页
        b：向上翻页
        /子串：向下搜索子串内容
        ?子串：向上搜索子串内容
        n：重复前一个搜索
        N：反向重复前一个搜搜
        q：离开这个less程序
        :f：显示文件名以及目前显示的行数

8：文件内容编辑:(vim)
    vim简介：
        vim是linux上一款文件内容编辑工具；
    vim模式：
        命令模式：打开或者新建一个文件时，默认模式即为命令模式，此模式下，所有输入内容都会作为命令，而非文件内容；
        输入模式：编辑文件内容时，即处于输入模式，此模式下，所有输入内容都会被作为文件内容；
        底线命令模式：对文件本身进行管理操作时的模式，此模式下，输入的内容均会被当做底线命令；而非文件内容；
    vim模式切换：
        命令模式：
            1：新建一个文件或者打开一个文件时，此时的默认模式就是命令模式；
            2：在输入模式按下ESC按键，即可进入命令模式；
            3：在底线命令模式下按下ESC按键，即可进入命令模式；
        输入模式：
            1：在命令模式下按下a[A]/i[I]/o[O]即可进入输入模式；
            2：在底线命令模式下，需要先进去命令模式，然后再按a[A]/i[I]/o[O]按键进入输入模式；
        底线命令模式：
            1：在命令模式输入 : 即可进入底线命令模式；
            2：在输入模式需要先退回命令模式，然后输入 : 进入底线命令模式；
    vim操作命令：
        1：命令模式下相关操作：
            1：模式切换：
                a[A]/i[I]/o[O]：进入输入模式
                :：进入底线命令模式
                [ESC]：由输入模式或者底线命令模式进入命令模式
            2：光标：
                H：光标移动至当前屏幕上最上方一行的首个字符；
                M：光标移动至当前屏幕上中间一行的首个字符；
                L：光标移动至当前屏幕上最下方一行的首个字符；
                G：光标移动至文件的最后一行；
                nG：光标移动至文件的第N行；
                gg/1G：光标移动至首行；
                n[enter]：表示光标向下移动n行；
                [end]：光标移动至当前行最后一个字符；
                [Home]：光标移动至当前行首个字符；
                [ctrl+b]：向后翻页；
                [ctrl+f]：向前翻页；
                [ctrl+e]：向下滚动；
                [ctrl+y]：向上滚动；
                [ctrl+d]：向下滚动半屏；
                [ctrl+u]：向上滚动半屏；
                [n<space>]：光标向右移动几个字符；
                w/W：移动到下一个单词的开头；
                b/B：移动到上一单词的开头；
                e/E：移动到光标所在单词的末尾；
                j：下移一行；
                k：上移一行；
                :n [enter]：移动至第n行；
            3：内容：
                删除：
                    x：删除当前光标后面的一个字符
                    X：删除当前光标前面的一个字符
                    nx：从当前光标处向右连续删除n个字符
                    nX：从当前光标处向左连续删除n个字符
                    dd：删除光标所在行
                    ndd：从光标所在行处，向下删除n行(包括当前行)；
                    dnG：删除光标所在行到第n行的数据；
                    dG：从光标所在行删除到最后一行；
                    d$：删除光标所在处到该行最后一个字符；
                    d0：删除光标所在处到该行最开始的字符；
                复制：
                    yy：复制光标所在行；
                    nyy：从光标所在行向下复制n行；
                    yG：复制光标所在行到最后一行的所有内容；
                    ynG：复制光标所在行到第n行的所有数据；
                    y0：复制光标所在位置到该行的首字符；
                    y$：复制光标所在位置到该行末尾的字符；
                粘贴：
                    p：将已复制内容粘贴在下一行
                    P：将已经复制的数据粘贴在上一行
                    u：回退前一个操作
                    .：重复上一个操作

        2：底线命令模式：
            1：行号：
                set number/set nu：显示行号
                set nonumber/set nonu：隐藏行号
            2：状态修改：(需要先输入:进入底线命令模式)
                w：保存编辑的内容
                w!：强制保存文件编辑的内容
                q：退出文件，并放弃对文件的修改
                q!：强制退出文件，并放弃对文件的修改
                x：保存文件之后退出文件
                w new_name：将文件保存成另一个文件
                n1,n2 w new_name：将n1行到n2行的内容保存成另一个文件
            3：查找和替换：
                /word：在文档中从当前光标处向下查询word字符串
                ?word：在文档中从当前光标处向上查询word字符串
                n：进入查询状态之后按 n 向下重复上次一次查询
                N：进入查询状态之后按 N 向上重复上一次查询
                [:n1,n2s/word1/word2/g]：n1和n2为数字，代表行号；该命名意为在n1和n2之间查找word1，并替换为word2;
                [:n1,$s/word1/word2/g]：n1为数字，代表行号；该命名意为在n1和文件末尾之间查找word1，并替换为word2;
                [:n1,$s/word1/word2/gc]：n1为数字，代表行号；该命名意为在n1和文件末尾之间查找word1，并替换为word2，并且在替换之前提示用户是否进行替换;
        3：输入模式：
            1：输入模式进入退出：
                1：a            :从当前光标位置的下一个字符插入
                A            :从当前光标所在行的最后一个位置插入
                
                2：i            :从目前光标所在处的前一个字符开始插入
                I            :从目前光标所在行的第一个非空字符开始插入

                3：o            :在目前光标所在行的下一行插入新的一行
                O            :在目前光标所在行的上一行插入新的一行
                
                4：r            :取代光标所在位置字符
                R            :一直取代光标所在位置字符，直到按下ESC为止；

                5：ESC           :退出输入模式，进入命令行模式
            2：光标移动：
                方向键：移动光标
                HOME/END：将光标移动到行首/行尾
                Page Up/Page Down：上下翻页
