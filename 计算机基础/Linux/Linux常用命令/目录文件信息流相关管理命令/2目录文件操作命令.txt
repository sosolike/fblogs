1：新建文件/目录:(mkdir)
    1.1：linux中并无直接创建空白文本文件的命令。可以间接使用文本编辑工具:vim或者信息流重定向命令:>、>>来创建文件；
    1.2：新建目录:mkdir
        mkdir [-m 权限值][-p] 待创建目录路径
        功能：创建指定目录路径
        参数：-m：指定待创建目录权限值；不指定该参数时，默认为系统默认权限；
             -p：递归创建目录；不指定该参数时，默认不允许递归创建目录；即只能创建路径最后一级目录；
        常见使用方式：
            1：创建一个目录，权限为系统默认权限：
                mkdir /tmp/tmpa/tmpb     #要求路径/tmp/tmpa必须已经存在，并且对该用户有写入和执行权限；
            2：创建一个目录，并指定相关权限：
                mkdir -m 755 /tmp/tmpa/tmpb     #要求路径/tmp/tmpa必须已经存在，并且对该用户有写入和执行权限；
            3：递归创建目录：
                mkdir -p /tmp/tmpa/tmpb     #只需要根路径存在即可，不要钱tmp/tmpa/tmpb存在，即可递归创建上述三级目录；
            4：递归创建目录并指定权限：
                mkdir -m 755 -p /tmp/tmpa/tmpb      #递归创建目录的同时指定权限，该权限仅对最后一级目录有效；

2：删除文件/目录:(rm/rmdir)
    rm -[rif] 待删除文件/目录路径
    功能：删除文件或者目录
    参数：-r：删除目录，不指定该参数时仅能够删除单个文件；
            -i：删除时是否询问确认，默认删除时会询问；
            -f：是否强制删除，默认会询问确认之后删除，如果指定该参数，将不再询问，直接删除；
    常见使用方式：
        1：删除一个文件，并询问：
            rm -i /tmp/a.txt  或者  rm /tmp/a.txt
        2：强制删除一个文件：
            rm -f /tmp/a.txt
        3：删除一个目录并询问：
            rm -r /tmp/tmpa
        4：强制删除一个目录：
            rm -f /tmp/tmpa
    
    rmdir [-p]
    功能：删除空目录
    参数：-p：递归删除空目录
    常见使用方式：
        1：删除指定空目录：
            rmdir /tmp/tmpa/tmpc      #要求tmpc必须是空目录
        2：递归删除空目录：
            rmfir -p /tmp/tmpa/tmpb     

3：复制文件/目录:(cp)
    cp [-dpRafil] 源路径 目标路径
    功能：将源路径内容复制至目标路径
    参数：-d：复制时保留链接
         -p：在复制文件内容的同时复制该文件的修改时间和访问权限；
         -R：复制目录，不带该参数时仅能够复制文件；
         -a：等同于同时携带-dpR参数；
         -i：如果目标文件/目录已经存在，则将询问是否继续进行复制操作；
         -f：强制赋值，如果目标文件已经存在也强制覆盖，不再询问；
         -l：仅生成源文件的链接，并不实际复制文件内容；
    常见使用方式：
        1：复制单个文件：
            cp 源路径 目标路径
        2：复制目录：
            cp -R 源路径 目标路径
        3：强制复制目录：
            cp -Rf 源路径 目标路径

4：移动文件/目录:(mv)
    mv [-ifu] 源路径 目标路径
    功能：将文件/目录移动至其他位置,并允许修改移动之后的文件/目录名称；
    参数：-i：如果目标文件/目录已经存在，询问是否移动该文件/目录；
         -f：强制移动。及时目标文件/目录已经存在，仍然移动该文件/目录；
         -u：如果目标文件已经存在，并且比源文件修改时间更早，才会移动当前文件并覆盖旧文件；
    常见使用方式： 
        1：移动文件至其他位置,移动时询问确认：
            mv /tmp/tmpa/a.txt /tmp/tmpb/b.txt
        2：移动目录至其他位置，移动时询问确认：
            mv /tmp/tmpa /tmp/tmpb
        3：强制移动文件，不再询问确认：
            mv -f /tmp/tmpa/a.txt /tmp/tmpb/b.txt
        4：仅在目标文件比当前文件修改时间更早时移动：
            mv -u /tmp/tmpa/a.txt /tmp/tmpb/b.txt
        5：重命名当前文件/目录：
            mv /tmp/tmpa/a.txt /tmp/tmpa/b.txt
            mv /tmp/tmpa /tmp/tmpb

5：修改文件/目录所属用户:(chown)
    chown [-R] 所属用户名(或：所属用户id) 待修改路径
    功能：将目标文件/路径所属用户修改为指定用户；
    参数：-R：递归设置目录中子目录以及子文件所属用户，默认只能设置单个文件或者目录本身所属用户；
    常见使用方式：
        1：设置目录本身所属用户：
            chown user /tmp/tmpa    #只能设置tmpa这个目录本身所属用户，目录中的子目录以及文件不受影响；
        2：设置文件所属用户：
            chown user /tmp/tmpa/a.txt
        3：递归设置目录本身以及目录中所有文件和所有子目录所属用户：
            chown -R user /tmp/tmpa

6：修改文件/目录所属用户组:(chgrp)
    chgrp [-R] 所属用户组名(或：所属用户组id) 待修改路径
    功能：将目标文件/路径所属用户修改为指定用户组；
    参数：-R：递归设置目录中子目录以及子文件所属用户组，默认只能设置单个文件或者目录本身所属用户组；
    常见使用方式：
        1：设置目录本身所属用户组：
            chgrp user /tmp/tmpa    #只能设置tmpa这个目录本身所属用户组，目录中的子目录以及文件不受影响；
        2：设置文件所属用户组：
            chgrp user /tmp/tmpa/a.txt
        3：递归设置目录本身以及目录中所有文件和所有子目录所属用户组：
            chgrp -R user /tmp/tmpa

7：修改文件/目录所属权限:(chmod)
    chmod [-R] 所属权限值(或：权限标识) 待修改路径
    功能：将目标文件/路径所属用户修改为指定权限；
    参数：-R：递归设置目录中子目录以及子文件所属权限，默认只能设置单个文件或者目录本身所属权限；
    说明：文件/目录权限包括所属用户权限，所属用户组权限及其他用户权限；权限本身包括读权限，写权限，执行权限；
    常见使用方式：
        1：设置目录本身所属权限：
            chmod 755 /tmp/tmpa    #只能设置tmpa这个目录本身所属权限，目录中的子目录以及文件不受影响；
        2：设置文件所属权限：
            chmod 755 /tmp/tmpa/a.txt
        3：递归设置目录本身以及目录中所有文件和所有子目录所属权限：
            chmod -R 755 /tmp/tmpa

9：进入目录:(cd)
    cd 目录路径
    功能：进入目录
    参数：无
    常见使用方式：
        1：进入本级目录下的子目录dir：
            cd ./dir   
        2：进入上级目录：
            cd .. 或者 cd ../
        3：进入根目录：
            cd /
        4：进入当前用户home目录：
            cd ~

            cd /dir1/dir2/dir3

10：查看目录信息：
    ls [-alrtARh] [目录路径]
    功能：查看指定目录下文件列表
    参数：-a：查看所有文件以及子目录，默认只显示非隐藏目录/文件，-a参数可以显示包括隐藏文件/目录在内的所有文件/目录；
         -l：以详细列表的方式展示该目录下的子目录和文件，默认只显示子目录/文件的名称；
         -r：将子目录/文件以相反顺序显示，默认以英文字母的顺序排序；
         -t：将子目录/文件以建立时间的顺序显示，默认以英文字母的顺序排序，rt参数带上，则以建立时间按照相反的顺序排序；
         -A：和-a参数相同，但是不列出.和..目录；
         -R：递归列出子目录，默认只列出当前目录中的子目录/文件信息；
         -h：以-l详细模式列出目录/文件信息时，文件/目录大小参数会以易读的方式显示；
         目录路径：默认列出当前所在目录的子目录/文件信息，如果带上目录路径参数，则列出给定目录的信息；
    其他：ll：等同于ls -l;对于ll命令也可使用上述参数跳转显示格式；
    常见使用方式： 
        1：简单查看目录中的信息：
            ls  或者 ls 指定目录路径
        2：查看目录中所有子目录/文件信息：
            ls -a 或者 ls -a 指定目录路径
        3：以详细格式查看目录中所有子目录/文件信息： 
            ls -al  或者 ls -al 指定目录路径
        4：安装建立时间顺序显示目录中的信息：
            顺序：ls -alt  或者 ls -alt 指定目录路径
            逆序：ls -altr 或者 ls -altr 指定目录路径
        5：只查看当前目录中的目录：
            ls -al | grep "^d"   或者    ll -al | grep "^d"
        6：查看当前目录中的子目录/文件数量：
            ls | wc -l
        7：查看当前目录中的文件，并以文件大小逆序排序：
            ls -al | grep "^-"|sort -k5nr
        8：查看当前目录中的文件，并以文件大小逆序排序，且只查看前10个：
            ls -al | grep "^-"|sort -k5nr | head -n 10

12：查看目录/文件类型：(file)
    file [-bizL] 目标路径
    功能：查看指定路径结点文件类型
    参数：-b：忽略文件名
         -i：打印文件的mini类型
         -z：解读压缩文件内容
         -L：直接显示符号链接所指向的文件类型
    常见使用方式：
        1：查看文件类型：
            file /a.txt
        2：查看文件mini类型：
            file -i /a.txt

13：创建目录/文件链接：(ln)
        ln [-bdfinsv] [源文件或目录] [目标文件或目录]
        功能：为某一个目录/文件在另外一个位置创建一个链接；包括软链接和硬链接；
        软硬连接：
            软链接：
                1.软链接，以路径的形式存在。类似于Windows操作系统中的快捷方式
                2.软链接可以跨文件系统，硬链接不可以
                3.软链接可以对一个不存在的文件名进行链接
                4.软链接可以对目录进行链接
            硬链接：
                1.硬链接，以文件副本的形式存在。但不占用实际空间。
                2.不允许给目录创建硬链接
                3.硬链接只有在同一个文件系统中才能创建
        参数：
            -b  删除，覆盖以前建立的连接；
            -d  允许超级用户制作目录的硬链接
            -f  强制执行
            -i  交互模式
            -n  把符号链接视为一般目录
            -s  软连接
            -v  显示详细处理过程
        常见使用方式：
            1：创建目标文件硬链接：
                ln 源文件 创建的连接文件

14：查看目录/文件大小：(ls/du)
    1：查看目录/文件占用磁盘大小
        du [-ashclL] [文件/目录]
        功能：查看文件/目录占用的磁盘空间大小，du命令所查大小并非文件实际占用的字节数，而是根据文件占用的磁盘io block数量计算而出；
        参数： 
            文件/目录：默认查询当前目录占用磁盘大小，也可以指定目录/文件路径查看指定目录/文件大小；
            -a：显示目录下所有子目录和文件占用磁盘空间大小；
            -s：显示目录占用磁盘大小时，不要分别显示该目录下子目录和文件占用的磁盘空间大小；和-a相反；
            -c：显示指定的多个文件/目录占用磁盘空间的大小，并统计他们的总和；
            -h：以可读的方式显示；即带上大小单位如：GB、KG、MB
            -l：统计硬链接占用磁盘空间大小；
            -L：统计符号链接所指向的文件占用磁盘空间大小；
        说明：
            1：du命令检测的文件大小实际是文件占用的磁盘io block数量*磁盘io block大小；linux下默认一个io block大小是4kb；
            2：如果查看文件实际字节数，需要使用ls -l或者ll来查看；
        常见使用方式：
            1：显示当前目录占用磁盘空间大小,并统计子目录占用磁盘空间大小：
                du -ah ./
            2：显示当前目录占用磁盘空间大小，不显示该目录下子目录占用磁盘空间的大小：
                du -sh ./
            3：显示指定几个文件占用磁盘空间大小，并计算总和：
                du -ch ./a.txt ./b.txt
            4：按照文件占用磁盘大小逆序显示目录中的文件信息：
                du -a|sort -k1nr
            
    2：查看目录/文件实际大小:(ls/ll):
        1：查看大小字节数：ls -l 或者 ll,即可以详细模式查看目录/文件实际大小；
        2：以易读模式查看大小：ls -lh 或者 ll -h,即可以易读模式查看目录/文件大小；
