1：管道命令:|
    管道符号：进程1 | 进程2
    管道命令功能：
        将文件信息或者标准输出信息发送至管道之后的命令进行处理
    管道说明：
        1：管道命令是linux在shell层面提供的一种进程之间单向通信的机制；使用管道会将前一个进程的标准输出传递至管道后的进程；
        2：管道对于信息的传递是单向的，只能将前一个进程的输出信息传递至管道后的进程；
        3：进程将消息传递至管道之后即结束，不会再打印至标准输出；
        4：管道中的消息读取是一次性的，不可重复读取；
        5：管道的右边进程自身必须能够接收标准输入流的能力，才能够接收管道的内容；
        6：管道不传递前一个进程的标准错误信息，只传递标准输入信息；

2：重定向命令：> >> <
    >：将标准输出和标准错误内容重定向至其他存储设备(文件等)，将会覆盖目标文件原内容；
    >>：将标准输出和标准错误内容重定向至其他存储设备(文件等)，将会追加在原文件内容之后；
    <：将数据来源为标准输入修改成文件或者其他设备的输入；

3：文本内容数量统计：wc
    wc [-lcw] [文件]
    功能：统计文件或者输入wc进程文本信息的数量，可以统计行数、字数、字节数等
    参数：
        l：只显示行数
        w：只显示字数(以空格作为分隔符)
        c：只显示字节数
    说明：
        1：默认情况下，wc会显示行数、字数、字节数
        2：wc除过统计文件内容以外，还可以紧跟管道命令，统计管道输入至wc命令的信息；

4：cut命令：
    功能：对文件以行为单位用指定的分割符进行分割，并选取所需要的字段；一般用在管道命令|后面，用来对内容进一步处理；
    参数：
        -c 以字符为单位分割；使用方式类似-f
        -b 以字节为单位进行分割；使用方式类似-f
        -d 用于定义分割符；
        -f 用来选取字段：
            取法有：
                N：只取第N项：-f 2;
                N-：从第N项一直到行尾；-f 2-;
                N-M：从第N项一直到第M项；-f 2-4;
                -M：从第一项一直到第M项；-f -3;
        (-f必须和-d一起连用)
        -n 取消分割多字节字符，仅和-b连用

5：文本信息排序：sort
    sort命令用于对文本文件内容以行为单位排序，但是其数据来源不仅仅支持文本文件，也包括标准输入和其他命令输出；
    语法：
        sort [-bcdfimMnrk][-o<输出文件>][-t<排序列分割字符>][文件]
        功能：对文本文件内容、标准输出内容按行排序
        参数：
            常用参数：
                M：将前面3个字母依照月份的缩写进行排序
                n：依照数值大小排序
                o：将排序后的结果存入指定文件
                r：以相反的顺序来排序
                t：指定排序时所用的栏分隔字符
                k：指定按照第几栏排序
                b：忽略每行前面开始出现的空格字符
            其他参数：
                c：检查文件是否已经按照顺序排序
                d：排序时仅处理英文字母、数字、空格字符；忽略其他字符
                f：排序时将小写字母视为大写字母
                i：排序时。忽略ASCII字符中040至176以外的其他字符
                m：将几个排序后的文件进行合并
            
        使用说明：
            1：指定栏位分隔符：
                使用说明：排序时可以按照栏位排序，因此指定排序栏位之前需要先确定以何种方式来将一行数据分隔成多个栏位；
                使用语法：-t "栏位分隔符"，默认以空格作为分隔符；
                使用示例：sort -t ":" ./tmp.txt

            2：排序类别：
                使用说明：在对数据行排序时默认是按照字符顺序排序的，也可以将字符指定为月份或者数字排序；
                使用语法：
                    按照数字排序：携带参数：n
                    将开始的三个字符识别为月份：携带参数：M
                使用实例：
                    按照字符排序：sort ./tmp.txt
                    按照数字排序：sort -n ./tmp.txt
                    识别为月份排序：sort -M ./tmp.txt
            
            3：指定排序栏位：
                使用说明：默认情况下按照一行的所有字符排序，也可将数据行分隔为多个栏位，并按照指定的栏位排序：
                使用语法：
                        指定栏位：-km 指定按照第m栏开始比较
                        指定栏位和字符：-km.w 指定从第m栏的第w位之后开始比较
                        指定多个栏和字符：-km.w,p.w 指定从第m栏的w位开始比较一直到第p栏的第w位；
            
            4：将排序结果输出至另一个文件：
                使用说明：默认排序结果将会打印至标准输出，可以使用o参数将排序结果打印至另一个文件中
                使用语法：sort -o ./aim.txt ./tmp.txt

        使用实例：
            对于文件a.txt，其内容为：
                1:23
                56:7
                2:34
                4:56
                33:3
                
            1：直接按照整行字符排序：
                sort ./t.txt   结果：
                                    1:23
                                    2:34
                                    33:3
                                    4:56
                                    56:7
                说明：默认以字符的形式按照整行排序；

            2：按照:分隔，并按照第2栏整体以数字方式逆序排序：
                sort -t ":" -k2nr ./t.txt  结果：
                                                4:56
                                                2:34
                                                1:23
                                                56:7
                                                33:3
                说明：
                    -n参数指定以数字的方式而非字符的方式排序
                    -r参数指定逆序排序
                    -k参数指定安装第2栏排序
            
            3：按照:分隔，并按照第2栏的第1个字符以数字方式逆序排序：
                sort -t ":" -k2.1,2.1nr ./a.txt 结果：
                                                    56:7
                                                    4:56
                                                    2:34
                                                    33:3
                                                    1:23
                说明：
                    k参数不仅可以指定排序栏位，也可以精细至对应栏位的第几位
